# מערכת פוליטית ישראלית · מחברת למידה

מחברת אינטראקטיבית לקורס "מערכת פוליטית ישראלית" — פרופ' דורון נבות, אוניברסיטת חיפה, תשפ"ו ב'.

11 יחידות לפי הסילבוס, מבוססות על מאמרי החובה.

## Stack

- Vite 8 + React 19 + TypeScript
- Tailwind 3 (RTL + dark mode)
- framer-motion + lucide-react

## Dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # → dist/
```

## Deploy

Vercel — `political-system-app.vercel.app`. Auto-deploys from `main`.
