## הבעיה
ביחידות, מילים מודגשות עם **קו מקווקו** פותחות tooltip עם הגדרת מושג. ה-tooltip מציג את `definition` מ-`src/content/units/unitNN/keyTerms.ts` (הרכיב `RichLine` ב-`components/common/RichText.tsx` מזהה את ה-`triggers` של כל מושג בטקסט הפסקה ועוטף אותם).

**שתי תקלות:**
1. **כפילות** — הרבה `definition` פשוט חוזרים על מה שכבר כתוב בפסקה הצמודה (לרוב הסבר בסוגריים). ה-tooltip לא מוסיף ערך.
2. **טעויות מקור שעברו ל-keyTerms** — אחרי סקירת התוכן (PRs #31–#36), התברר ש-keyTerms נשאו את אותן טעויות שתוקנו בסקציות, והן **live ב-tooltips**.

### שיטה (מוכחת על unit07)
לכל יחידה:
1. **זהה את הכפולות עם Gemini (טקסט, לא תמונות!):**
   ```bash
   cd "/c/Users/Ori-PC/.gemini/tmp/playwright"   # סנדבוקס — קבצים חייבים להיות פה
   # העתק את keyTerms.ts ואת קבצי הסקציות של היחידה לתיקייה משותפת, ואז:
   gemini -p "הקובץ @keyTerms.ts מגדיר מושגים (definition). הסקציות @01-....ts ... מציגות tooltip עם ה-definition כשמילת trigger מופיעה. לאילו מושגים ה-definition רק חוזר על מה שכבר כתוב בפסקה הצמודה (כפל מיותר)? ציין id + למה. אם definition מוסיף ערך — דלג. עברית, רשימה ממוקדת."
   ```
2. **שכתב כל כפולה** כך שתוביל ב**פואנטה האנליטית / ההבחנה / השימוש של המחבר** — לא חזרה על הפסקה. שמור את `term` + `english` + `triggers`, שנה רק את `definition`. ראה את unit07/keyTerms.ts כסטנדרט.
3. **תוך כדי — בדוק דיוק:** keyTerms עלולים לשאת טעויות מקור. אמת כל טענה/תאריך/מספר/ייחוס מול המאמר (טקסט נקי — ראה למטה). תקן.
4. `npx tsc --noEmit` + `npm run build` + `npm run verify:bundle` → branch + PR + merge פר יחידה (או באצ' של 2-3). אחרי merge: ודא deploy (HTTP 200).

### כללי איכות לשכתוב הגדרה
- ההגדרה צריכה להוסיף משהו שהקורא **לא** קיבל מהפסקה: הנקודה האנליטית, הבחנה מדויקת, או השימוש הספציפי של המחבר במושג.
- לא להמציא — להישען על המאמר.
- **אסור לתת ל"מקרוב" trigger** (זו גם מילה עברית רגילה = "לאחרונה/מקרוב" → תתפוס שגוי).
- לשמור על קונבנציית code-splitting (אין נגיעה ב-loaders/UNIT_METAS לשינויי keyTerms בלבד).

### טקסטים נקיים של המאמרים (לאימות דיוק)
- אנגלית / עם שכבת טקסט → `pdftotext` ישיר.
- עברית סרוקה → גרסת **תלם** (`...תלם.pdf`) או **כותר** (`*_OCR.pdf`). **נתיבי עברית שוברים bash → PowerShell + העתקה לשמות ASCII** לסנדבוקס `C:\Users\Ori-PC\.gemini\tmp\playwright\`.
- אל תיתן ל-Gemini תמונות עברית צפופות (מתבלבל על RTL וקורס על quota).
