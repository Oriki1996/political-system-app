# חומרי לימוד — מערכת פוליטית ישראלית

**קורס:** פרופ' דורון נבות · אונ' חיפה · תשפ"ו ב'
**אתר אינטראקטיבי:** https://political-system-app.vercel.app

תיקייה זו מכילה את החומרים המקוריים (OCR של מאמרי החובה) + סיכומים לכל יחידה, מסודרים ללמידה אישית ולעבודות בית.

## 📘 חומר לימוד מלא (DOCX)

`מערכת_פוליטית_ישראלית__חומר_לימוד_מלא.docx` — ספר לימוד עברי RTL יחיד ובו **כל תוכן הסקציות של 10 היחידות** (כותרת, מאמרי חובה, שאלת המנחה, מטרות, פסקאות, טבלאות השוואה, ציטוטים, הדגשות) — **ללא שאלות ותרגול**. לקריאה ולמענה על מטלות הקורס.

לבנייה מחדש אחרי עדכוני תוכן: `node _scripts/extract_study_content.cjs && python _scripts/build_study_docx.py`.
