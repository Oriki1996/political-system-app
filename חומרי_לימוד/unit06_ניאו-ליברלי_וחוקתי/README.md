## מושגי מפתח שאסור לבלבל

- **המהפכה החוקתית** (1992-1995) — חוקי-יסוד + פסיקת ברק בבנק המזרחי
- **מהפכת המשפט** (שנות ה-80) — אקטיביזם שיפוטי שקדם לחוקי-היסוד
- **ניאו-ליברליזם קוסמופוליטי** = יוני, מערב-אירופי, גרמני (קוהל, פרידמן)
- **ניאו-ליברליזם בדלני** = נצי, אנגלו-אמריקני (תאצ'ר, רייגן, נתניהו)
- **חוק אי-ההדפסה** (1985) — בנק ישראל אסור להלוות לממשלה. תחילת עצמאות הבנק.
- **חוק יסודות התקציב** (1985) — האוצר ואגף התקציבים שולטים בתקציב.
- **חוק ההסדרים** (1985) — רפורמות בחקיקה אחת, עוקפת דיון פרטני.

## טיפים לעבודת בית

- **שאלת הסינתזה:** מה הקשר בין המהפכה החוקתית למפנה הניאו-ליברלי? **תשובה:** שניהם מעבירים כוח מהפוליטיקה למומחים — שופטים וכלכלנים.
- **שאלת קרמפף:** למה ישראל הקצינה דווקא אחרי 2000? **תשובה:** אינתיפאדה שנייה → אי-הסכמה בינלאומית → צורך בעצמאות כלכלית → מודל בדלני.
- **שאלת הטעיה (ספיר):** האם לין שיקר? **תשובה (גרסה מרוככת):** לין אולי האמין למה שאמר, אבל ברק פירש בדיעבד הפוך → "הטעיה דרך זמן".
